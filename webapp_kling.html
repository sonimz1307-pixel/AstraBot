Провайдер

Выбери провайдера, затем настройки и нажми “Сохранить”.

Тип генерации

Для Motion Control длительность задаётся референс-видео (3–30 сек).

Качество

Standard — 1 сек = 1 токен (≈9–10 ₽)
Pro — 1 сек = 2 токена (≈18–20 ₽)

Длительность

Длительность применяется только для Image → Video.

Модель Veo

Fast: без звука 1 токен 1сек, со звуком 2 токена 1сек.

Режим

Промпт ты введёшь в боте, WebApp сейчас задаёт только параметры.

Параметры

Генерировать звук
Дороже, но “как ролик”
Доп. настройки (только Veo 3.1)
Эти опции доступны только в Veo 3.1. Выбери Veo 3.1 сверху.
Last frame (финальный кадр)
Попросим в боте 2-ю картинку
Reference images (до 3)
Попросим в боте 0–3 референса
Доступно только в Veo 3.1
