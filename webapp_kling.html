Сценарий из нескольких планов (запуск по промпту). В тексте используй планы/шоты.
Audio
Добавляет звук к ролику. Audio = +1 токен/сек к стоимости.

Audio

Multi-shot scenes

Заполни минимум 2 шота. Сумма 3–15 сек. Основной prompt для Multi-shot не нужен: используются только shot prompt. Image element можно загрузить прямо внутри каждого shot — на 1 element нужно минимум 2 фото и максимум 4 фото, имя вставится в prompt как @name.
После сохранения: Text → пришли prompt; Image → пришли стартовое фото, опционально второй кадр, затем prompt; Multi-shot → основной prompt не нужен, заполни только shot-блоки, при желании пришли общий стартовый кадр, затем отправь в боте сообщением: «СТАРТ». Elements уже будут загружены автоматически.
